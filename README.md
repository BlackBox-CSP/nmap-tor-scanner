### Standalone Nmap python app
```
    usage: ./nmap-tor.py <options>'
    options:'
        -h, --help        Display this message'
        -t, --target      Specify single IP address or network of target'
        -f, --targetlist  Specify file of IP addresses and/or networks to use as target'
        -p, --portlist    Specify file of ports to be used on target'
        -s, --sleep       Specify time in seconds to sleep between Nmap requests (default:10)'
        -n, --numhosts    Specify number of hosts to be scanned from the provided list (default:all)'
```
